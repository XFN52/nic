# Citizine Tracker (Ник)

Приложение на Flutter для отслеживания 25-дневного курса приема препарата "Цитизин" (средство для отказа от курения).

## 🚀 Описание проекта

**Citizine Tracker** — это персональный помощник, который помогает строго следовать схеме приема таблеток в течение всего курса. Приложение автоматически рассчитывает интервалы между приемами в зависимости от текущего дня курса и уведомляет о возможности приема следующей дозы.

### Основные функции:
- **Автоматический расчет схемы**: Интервалы меняются от 2 часов в первые дни до 5 часов к концу курса.
- **Таймер следующей дозы**: Наглядный индикатор того, сколько времени осталось до следующего приема.
- **История приемов**: Логирование каждой принятой таблетки.
- **Прогресс курса**: Интерактивная сетка на 25 дней, отображающая текущий статус.
- **Сброс курса**: Возможность начать курс заново с полной очисткой данных.

## 🛠 Технологический стек
- **Framework**: [Flutter](https://flutter.dev)
- **Language**: Dart
- **Storage**: `shared_preferences` (для локального хранения данных)
- **Localization**: `intl` (для форматирования времени)

## 📦 Установка и запуск

1.  **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/nic.git
    cd nic
    ```

2.  **Установите зависимости**:
    ```bash
    flutter pub get
    ```

3.  **Запустите приложение**:
    ```bash
    flutter run
    ```
